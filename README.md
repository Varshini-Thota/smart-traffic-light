# Smart Traffic Controller

## Overview
A smart traffic light controller using state machine logic and pedestrian crossing mode.

## Features
- Red, Yellow, Green sequencing
- Non-blocking timing using millis()
- Pedestrian button support
- Event-driven embedded logic

## Technologies Used
- Arduino
- Embedded C
- Wokwi Simulator

## Future Improvements
- Emergency override mode
- UART control
- Countdown timer
